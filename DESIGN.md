
## Data Types

| Datatype | Representation |
| -------- | -------------- |
| integer | int |
| floating point  | float |
| string | str |
| boolean | bool |
| json | json |
| array | arr |

## Arrays
To create an array of any type, use the syntax:
```
single_value [float] = 3.14
some_array [arr float] = [0.1, 0.2, 0.3]
```
